Etsy Search Engine

Date Completed:

November 2016

Project:

Here I resdesigned the Etsy search engine and accessed their API for a product search. Implementing the clean design and infinte scroll allows the user to focus on what's important--their quest for the perfect Etsy product! My idea for this project stemmed from wanting to have a simple and beautiful application for quick product searching. This was my first time working with an API, so my biggest challenge was getting my bearings and fully understanding all the API has to offer. To build off this project in the future, I would add the ability to search for certain shops and users.
